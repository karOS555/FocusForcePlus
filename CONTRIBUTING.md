# Contributing to FocusForce+

Thanks for your interest in the project!

## Contributions are paused for now

FocusForce+ is still in its early stages. The codebase is changing rapidly, core features are not yet implemented, and the overall structure is still being figured out. Opening the project to external contributions this early would mean a lot of rework for everyone involved.

I plan to properly open contributions as soon as the app is functional and has reached a stable base. At that point I'll put together a proper contribution guide, set up good-first-issue labels, and write up areas where help would be especially welcome.

## What you can do in the meantime

- **Watch the repo** to get notified when things progress
- **Open an issue** if you spot something wrong or have a feature idea. I read everything and will tag things for later even if I can't act on them right away
- **Share the project** with anyone who might find it useful

## When contributions open

Once the project is ready for contributors, this file will be updated with:

- Development setup instructions
- Code style guidelines
- Commit message conventions
- Pull request workflow
- Translation guide
- Areas where help is most needed

Thanks for your patience!
